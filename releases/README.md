## 安装步骤

1. 下载 `.difypkg` 文件到本地
2. 在Dify工作空间中进入 **工具** → **插件** → **安装插件**
3. 选择 **"从文件安装"**
4. 上传下载的 `.difypkg` 文件
5. 点击 **安装** 按钮
