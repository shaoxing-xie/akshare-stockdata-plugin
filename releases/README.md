## 安装说明

1. **下载插件包**：点击上方链接下载 `.difypkg` 文件
2. **在Dify中安装**：
   - 在Dify工作空间中进入 **工具** → **插件**→  **安装插件**
   -  **安装源**中选择 **"GitHub"**
   - GitHub 仓库，输入：https://github.com/shaoxing-xie/akshare-stockdata-plugin
   - 安装页面中，选择版本号和安装包
   - 点击  **下一步** 、**安装** 按钮
